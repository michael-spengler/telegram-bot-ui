# Telegram Bot UI Server

## Usage Example

```sh

deno run --allow-read --allow-net ui-server.ts

```

or via pm2

```sh

pm2 start  --interpreter="deno" --interpreter-args="run --allow-read --allow-net" ui-server.ts

```
